# 🚖 Cab Booking Data Analytics & Interactive Dashboard

An end-to-end data analytics project on ride-sharing (cab booking) data using Python.  
This project focuses on data cleaning, exploratory data analysis (EDA), interactive dashboard visualization, and fare prediction using machine learning.

---

## 📌 Project Overview

The goal of this project is to analyze cab booking data to understand:
- Customer behavior
- Trip patterns
- Revenue trends

An interactive dashboard is built to visualize key insights and enable data-driven decision-making.

---

## ⚙️ Tech Stack

- **Python**
  - Pandas
  - NumPy
  - Matplotlib / Seaborn / Plotly
- **Streamlit** (for interactive dashboard)
- **Machine Learning**
  - Linear Regression (Scikit-learn)

---

## 🚀 Features

- ✅ Data Cleaning & Preprocessing  
- ✅ Exploratory Data Analysis (EDA)  
- ✅ Interactive Dashboard with Filters  
- ✅ KPI Metrics (Revenue, Trips, Fare, etc.)  
- ✅ Trend & Comparative Analysis  
- ✅ Fare Prediction using Regression Model  

---

## 📊 Key Insights

- Distance and duration strongly impact fare  
- Surge pricing significantly increases revenue  
- Most trips are short-distance and low-cost  
- Peak hours show higher ride demand and pricing  

---

## 📈 Dashboard Highlights

- Revenue by ride type  
- Rides by city  
- Monthly trends (rides & revenue)  
- Fare and distance distribution  
- Interactive filters (city, ride type, date, fare range)

---

## 🧠 Machine Learning Model

- Implemented **Linear Regression** to predict fare  
- Achieved good accuracy with strong correlation between features like distance and duration  

---

## ▶️ How to Run the Project

1. Clone the repository:
```bash
git clone https://github.com/ShrutiDesai7/cab-booking-data-analytics-dashboard.git

2. Navigate to project folder
cd cab-booking-data-analytics-dashboard

3.Install dependencies
pip install -r requirements.txt

4.Run the applications
streamlit run app.py

```
